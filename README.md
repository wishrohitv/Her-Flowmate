# HerFlowmate 🌸
Your gentle, premium cycle and pregnancy companion built with Flutter.

A minimalist, privacy-first tracker designed to empower women with deep biological insights, clinical-grade pregnancy tracking, and calm aesthetics.

## ✨ Features
- **🤰 Pregnancy Dashboard**: A high-density, fragment-based UI with trimester-aware progress and milestone tracking.
- **📅 Clinical Accuracy (LMP)**: Pregnancy logic centered on the **Last Period (LMP)** with standard 280-day due date calculations.
- **🔄 Cycle Evolution Wheel**: Interactive visualization of your current biological phase.
- **💧 Advanced Hydration**: Smart tracking with a daily **15 Glass target** and intuitive logging.
- **📝 Daily Check-ins**: Log symptoms, moods, and energy levels for a holistic view of your health.
- **🛡️ Web Stability (Nuclear Phase)**: Optimized for Chrome with a fragment-first architecture that eliminates rendering assertions.
- **🔒 Privacy-First**: No data leaves your device. No accounts required.

## 🛠️ Installation
1. Clone the repository.
2. Run `flutter pub get`.
3. Run `flutter run -d chrome`.

## 🤝 Contributing
Contributions are welcome! We follow a **Fragment-First** architecture for stability. Please ensure:
1. All dashboard components are built as `StatelessWidget` fragments.
2. Run `dart format .` before pushing.
